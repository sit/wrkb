# Demolish

**Type:** Minor
**Family:** Inspiration

## Description

When within 550 range of a turret, change an attack over 3 seconds. Attacking the turret discharges the attack dealing 200 (+ 30% maximum health) bonus Adaptive Damage. (30 second cooldown)
